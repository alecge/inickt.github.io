Useful Programs and Servies

  • Ninite (Windows only) - This website provides an easy way to install some of the most popular free programs. I use it after a clean install of Windows, and use Chrome, 7-Zip, VLC, all runtimes, Reader DC, Essentials, and Malwarebytes. I add programs like iTunes, Spotify, Dropbox, and Google Drive depending on the needs of that computer or user.
  • Google Chrome or Mozilla Firefox - Two very good internet browsers. I prefer them to Internet Explorer and Safari, however the newest versions of each are fine for basic use. I use Chrome because of its integration into the Google ecosystem.
  • Malwarebytes (Windows only) - The best free anti-malware tool available. Run it once and a while. Not a replacement for an antivirus.
  • Microsoft Security Essentials (Windows 7 and Vista only) - Very good free antivirus. Lightweight and will not bug you with updates. Windows 8 and above has MSE built in.
  • Unchecky (Windows Only) - Prevents other unneeded software from being installed when installing other programs.
